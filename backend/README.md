# Teslo API

1. Clone repository

2. Install dependencies

   ```
   yarn install
   ```

3. Clone `.env.template` and rename it to **.env**

4. Mount Database

   ```
   docker-compose up -d
   ```

5. Run project

   ```
   yarn start:dev
   ```

6. Execute seed to populate DB

   ```
   localhost:3001/api/seed
   ```
